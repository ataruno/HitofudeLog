# HitofudeLog
指定したmarkdownファイルにVScodeのコマンドラインからテキストを追記する拡張機能です。  
なるべく楽に日々のログを取れるようにしたい思いで作成しました。  

エディタ画面でカーソル選択した箇所をコマンドラインへ流し込むことも可能です。  


HitofudeLog.Add
HitofudeLog.Send
HitofudeLog.Open
