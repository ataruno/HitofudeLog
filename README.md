# HitofudeLog
指定したmarkdownファイルにVScodeのコマンドラインからテキストを追記する拡張機能です。  
なるべく楽に日々のログを取れるようにしたい思いで作成しました。  

## 設定
ログを書き込みたいmarkdownファイルを設定のTargetPathへ絶対パスで記載してください。  
例：C:/Users/Desktop/日報.md  

## コマンド
Hitofudelog.Add ... コマンドライン経由でログを記載します。  
Hitofudelog.Send ... VScode上でカーソル選択した文字列をコマンドライン経由でログとして記載します。  
Hitofudelog.Open ... 書き込み先に設定したmarkdownファイルを開きます。
